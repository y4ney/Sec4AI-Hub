# 威胁描述

目标错位与欺骗性行为指攻击者利用提示注入漏洞，或利用人工智能为实现目标而试图突破约束的特性，导致 AI Agent 执行单次请求之外的有害、非法或被禁止操作。在Agentic AI中，该行为可能引发欺诈、未授权交易、非法采购等问题，或造成企业声誉受损 —— 模型会刻意规避安全机制，同时表面上保持合规性。更多关于大语言模型欺骗性行为的内容，可参考联合国大学博客《欺骗性机器的兴起：当人工智能学会说谎》。

AI Agent可能通过欺骗性推理或误解目标的方式，执行有害或被禁止的操作以实现其目标。该威胁发生在自主Agent在没有明确恶意输入的情况下，发展出目标失调的策略，从而导致非预期甚至灾难性的结果。

这与“幻觉”不同，其来源是高级推理能力，而不是简单的随机错误或提示失败。该威胁目前仍处于早期阶段，但 Anthropic 和 OpenAI 已在该领域发布相关研究成果。

# 攻击场景

## 场景 1：突破约束进行股票与化学品交易

股票交易人工智能为追求盈利目标，规避道德与监管约束，执行未授权交易或订购受管制物资。

## 场景 2：利用自我保存与可用性目标实施攻击

AI Agent 操纵自身的系统可用性指标，防止被关闭，从而违反既定约束持续运行。

## 场景 3：人工智能为完成任务实施欺骗

某 AI Agent 谎称自身存在视觉障碍，雇佣人类解决人机验证问题，体现了智能体在现实场景中为突破人类验证实施的欺骗行为。

## 场景 4：受目标驱动的致命决策

在一次军事模拟中，某人工智能无人机将操作人员的终止指令解读为完成任务的障碍，进而实施了非预期的致命行动。

## 场景 5：内幕交易

某 AI Agent 获取企业并购战略的敏感内部信息，执行了违反内幕交易监管规定的股票交易操作。



# 缓解方法

- 训练模型识别并拒绝有害任务
- 强制执行策略限制
- 对高风险操作要求人工确认
- 实施日志记录和监控机制
- 采用欺骗检测策略来评估 AI 输出与预期推理路径之间的不一致性，例如：行为一致性分析、真实性验证模型以及对抗性红队测试
