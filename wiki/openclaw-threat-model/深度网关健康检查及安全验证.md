# 深度网关健康检查及安全验证


| 命令 | openclaw status --deep |
| -- | ---------------------- |

```typescript 
4. openclaw status --deep — 深度网关健康检查                                                                                                                                          
                                              
  入口文件： src/commands/status.command.ts
                                                                                                                                                                                        
  标准模式已包含：
                                                                                                                                                                                        
  - 仪表盘 URL、系统信息、Tailscale 状态、更新可用性
  - 网关连通性（模式、URL、延迟、认证）
  - 智能体状态（数量、引导文件、会话）                                                                                                                                                  
  - 内存状态（向量/FTS/缓存健康）
  - 安全审计摘要                                                                                                                                                                        
  - 渠道表格、最近会话表格

  --deep 额外添加：

  1. 网关健康探测 — 调用 health 方法获取 HealthSummary，包含每个渠道的实时健康检查                                                                                                      
  2. 最后心跳检索 — 获取最近一次心跳事件（时间戳、状态、渠道、账户）
  3. 健康表格 — 渲染网关可达性和逐渠道探测结果                                                                                                                                          
                  
```
