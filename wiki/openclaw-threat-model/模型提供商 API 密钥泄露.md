# 模型提供商 API 密钥泄露

#### 威胁描述

模型供应商的 API 密钥存储在配置文件中，这些文件可能具有过于宽松的文件系统权限。如果攻击者获得了文件系统的访问权限，就可以提取凭证，导致费用滥用或获得供应商完整的 API 能力。

#### 缓解措施

配置验证机制会拒绝具有全局可读权限的配置。安全审计会主动检查并报告全局可读的配置文件。凭证应尽可能使用操作系统的密钥链集成来存储。
