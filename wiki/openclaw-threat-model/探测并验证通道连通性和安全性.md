# 探测并验证通道连通性和安全性


| 命令 | openclaw channels status --probe |
| -- | -------------------------------- |

```typescript 
 3. openclaw channels status --probe — 渠道连通性探测
                                                                                                                                                                                        
  入口文件： src/cli/channels-cli.ts
  实现： src/commands/channels/status.ts                                                                                                                                                
                  
  工作流程：                                                                                                                                                                            
   
  1. 调用网关 channels.status 方法                                                                                                                                                      
  2. 网关可达时，显示每个渠道的：
    - 启用/禁用、已配置/未配置、已链接/未链接
    - 运行中/已停止、已连接/已断开
    - 最后一次入站/出站消息时间戳                                                                                                                                                       
    - Bot 用户名、DM 策略、allowFrom 列表
    - Token 来源及可用性                                                                                                                                                                
    - --probe 专属：主动测试每个渠道的凭证和连通性（而非仅报告缓存状态）                                                                                                                
    - 审计结果、最近错误                                                                                                                                                                
  3. 网关不可达时，回退到仅从配置文件读取状态 
```
