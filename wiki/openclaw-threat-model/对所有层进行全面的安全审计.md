# 对所有层进行全面的安全审计


| 命令 | openclaw security audit --deep |
| -- | ------------------------------ |

```bash 
 1. openclaw security audit --deep — 全面安全审计                                                                                                                                      
                                                                                                                                                                                        
  入口文件： src/cli/security-cli.ts                                                                                                                                                    
  核心实现： src/security/audit.ts → runSecurityAudit()                                                                                                                                 
                                                                                                                                                                                        
  标准模式（不加 --deep）执行约 30+ 项检查：                                                                                                                                            
                                                                                                                                                                                        
  ┌──────────────┬────────────────────────────────────────────────────────────────────────────────────────────┐
  │   检查类别   │                                          具体内容                                          │
  ├──────────────┼────────────────────────────────────────────────────────────────────────────────────────────┤                                                                         
  │ 网关配置     │ 绑定地址 vs 认证策略、CORS 白名单、Tailscale Funnel 暴露、可信代理配置、速率限制、令牌长度 │
  ├──────────────┼────────────────────────────────────────────────────────────────────────────────────────────┤                                                                         
  │ 文件系统权限 │ 状态目录权限（全局可写=严重）、配置文件权限、符号链接检测                                  │                                                                         
  ├──────────────┼────────────────────────────────────────────────────────────────────────────────────────────┤                                                                         
  │ 执行运行时   │ 沙箱模式一致性、exec 安全等级、解释器白名单、safeBins 风险路径                             │                                                                         
  ├──────────────┼────────────────────────────────────────────────────────────────────────────────────────────┤                                                                         
  │ 浏览器控制   │ CDP 认证要求、远程 CDP over HTTP、私有网络 CDP 目标                                        │
  ├──────────────┼────────────────────────────────────────────────────────────────────────────────────────────┤                                                                         
  │ 渠道安全     │ 逐渠道审计：DM/群组策略、allowFrom 列表、凭证暴露                                          │                                                                         
  ├──────────────┼────────────────────────────────────────────────────────────────────────────────────────────┤                                                                         
  │ 日志         │ 脱敏是否被禁用                                                                             │                                                                         
  ├──────────────┼────────────────────────────────────────────────────────────────────────────────────────────┤                                                                         
  │ 配置卫生     │ 配置中的明文密钥、模型风险、攻击面汇总                                                     │                                                                         
  └──────────────┴────────────────────────────────────────────────────────────────────────────────────────────┘                                                                         
                                                                                                                                                                                        
  --deep 额外执行：                                                                                                                                                                     
                                                                                                                                                                                        
  1. 插件和技能代码安全扫描 — 扫描所有已安装插件和技能的代码，检测危险模式                                                                                                              
  2. 实时网关探测 — 连接网关执行活体健康检查，验证认证和连通性
                                                                                                                                                                                        
  另有 --fix 选项可自动修复文件权限等问题。                                                                                                                                             
                                                  
```
