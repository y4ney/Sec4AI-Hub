# 沙箱逃逸

#### 威胁描述

Docker 沙箱可能通过卷挂载、容器逃逸漏洞或安全选项配置不当被绕过。成功逃逸后攻击者将获得宿主机系统访问权限。

#### 缓解措施

Docker 沙箱会验证安全选项。容器以最小权限运行（禁止 `--privileged`，禁止宿主机网络）。卷挂载受到限制。安全配置文件（seccomp、AppArmor）限制系统调用。
