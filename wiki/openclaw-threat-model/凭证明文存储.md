# 凭证明文存储

#### 威胁描述

OAuth 令牌、API 密钥和配对凭证以未加密的形式存储在 JSON 文件中。攻击者一旦获得文件系统访问权限，就能读取所有已存储的凭证，对所有已配置的渠道实施账户接管和数据窃取。

#### 缓解措施

配对存储使用有效期为 1 小时的短期验证码，并将每个渠道的待处理请求限制为 3 个。建议集成操作系统密钥链。安全审计会标记全局可读的凭证目录。
