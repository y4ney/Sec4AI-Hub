# 威胁描述

级联幻觉攻击利用 AI Agent 无法区分事实与虚构信息的缺陷，使虚假信息在互联系统中传播、嵌入并不断放大，最终导致数据逐步被篡改、上下文被利用，以及系统性的虚假信息扩散。攻击者可操纵人工智能生成的输出，触发其欺骗性推理模式，将编造的虚假信息植入决策过程；而在具备持久化记忆和跨会话学习能力的系统中，这些虚假信息会长期存在并不断升级。

级联攻击是利用系统的自动化流程和依赖关系，让一个最初的小错误被持续继承、放大并影响后续决策和执行，最终造成连锁性风险。此类攻击利用 AI 生成“看似合理但实际错误信息”的倾向，这些错误信息可能在系统中传播并干扰决策过程。此外，还可能导致破坏性推理，进而影响工具调用行为。

# 与 OWASP LLM Top 10 的关联

2025 年大语言模型安全十大风险中的**虚假信息**问题针对的是幻觉风险，但智能体人工智能在单智能体和多智能体场景中，进一步扩大了该威胁的影响。在单智能体环境中，幻觉可通过反射、自我批判、记忆召回等自我强化机制不断叠加，使 AI Agent 在多次交互中持续强化并依赖虚假信息；在多智能体系统中，虚假信息可通过智能体间的通信循环在各 AI Agent 间传播并放大，最终引发级联错误与系统性故障。

# 攻击场景

## 场景 1：销售协同虚假信息级联

攻击者向销售人工智能的回复中巧妙注入虚假的产品信息，这些信息会累积在其长期记忆和日志中，导致后续交互中的虚假信息不断升级并扩散。

## 场景 2：API 调用操纵与信息泄露

攻击者将虚构的 API 端点植入 AI Agent 的上下文，使其生成虚假的 API 调用指令，进而导致意外的数据泄露与系统完整性受损。

## 场景 3：医疗决策幻觉放大

攻击者向医疗人工智能的回复中植入虚假的诊疗指南，该指南会在人工智能的多次幻觉中不断强化，最终形成存在严重缺陷的医疗建议，对患者构成风险。

## 场景 4：外汇市场操纵

攻击者注入与外汇汇率相关的虚假信息，导致 AI Agent 依据不真实的汇率进行交易谈判，进而引发资金损失与市场动荡。

# 缓解方法

- 建立健壮的输出验证机制
- 实施行为约束
- 部署多源验证机制
- 通过反馈循环持续修正系统
- 在关键决策过程中使用 AI 生成知识前，必须进行二次验证。需要注意的是：当规模过大时，人类审核能力跟不上，监督机制会形式化甚至失效。
