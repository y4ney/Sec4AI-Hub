# 批准待处理的发送方配对请求


| 命令 | openclaw pairing approve \<channel> \<code> |
| -- | ------------------------------------------- |

```bash 
 6. openclaw pairing approve <channel> <code> — 批准配对

  入口文件： src/cli/pairing-cli.ts
  核心实现： src/pairing/pairing-store.ts → approveChannelPairingCode()

  工作流程：

  1. 获取配对存储的文件锁
  2. 清理过期请求
  3. 匹配验证码（大小写不敏感）                                                                                                                                                         
  4. 匹配成功：从待处理列表移除，将发送者 ID 写入 <channel>-<accountId>-allowFrom.json
  5. 若传入 --notify，通过渠道适配器向请求者发送确认消息                                                                                                                                
                  
  存储位置：                                                                                                                                                                            
  - 配对存储：<stateDir>/oauth/<channel>-pairing.json
  - 允许列表：<stateDir>/oauth/<channel>-<accountId>-allowFrom.json
  - 文件锁使用指数退避重试（10 次，100ms-10s）
```
