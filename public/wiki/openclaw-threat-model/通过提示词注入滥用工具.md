# 通过提示词注入滥用工具

#### 威胁描述

攻击者构造特定消息，触发 bash 或浏览器工具执行未授权的命令。成功注入后可以删除文件、窃取数据、安装恶意软件，或执行智能体工具所允许的任何操作。

#### 缓解措施

所有工具调用都会记录完整参数。工具使用模式受异常监控。沙箱配置包含允许列表和拒绝列表。提权命令需要显式审批。Bash 命令通过受限 Shell 运行。
