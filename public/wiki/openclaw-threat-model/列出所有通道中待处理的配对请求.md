# 列出所有通道中待处理的配对请求


| 命令 | openclaw pairing list |
| -- | --------------------- |

```typescript 
5. openclaw pairing list — 列出配对请求

  入口文件： src/cli/pairing-cli.ts
  核心存储： src/pairing/pairing-store.ts

  工作流程：                                                                                                                                                                            
   
  1. 读取配对存储文件 <credentialsDir>/<channel>-pairing.json                                                                                                                           
  2. 获取文件锁（带重试和陈旧检测）
  3. 清理过期请求（TTL = 60 分钟）
  4. 清理超额请求（最多 3 个待处理）
  5. 返回排序后的请求列表：id、code（8 位人类友好字母数字码）、createdAt、lastSeenAt、meta

  配对码规格：
  - 8 位字符，使用 29 字符字母表 ABCDEFGHJKLMNPQRSTUVWXYZ23456789（排除了易混淆的 0/O/1/I）
  - 每渠道最多 3 个待处理请求                                                                                                                                                           
  - 60 分钟过期
```
