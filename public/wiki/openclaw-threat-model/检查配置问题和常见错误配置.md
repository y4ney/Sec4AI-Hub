# 检查配置问题和常见错误配置


| 命令 | openclaw doctor |
| -- | --------------- |

```bash 
 2. openclaw doctor — 配置问题检查

  入口文件： src/commands/doctor.ts → doctorCommand()

  按顺序执行约 30 个检查步骤：

  1. 更新检查 — 是否有新版本可用
  2. UI 协议新鲜度 — Control UI 资源是否过期
  3. 配置加载与迁移 — 检测并应用配置格式迁移                                                                                                                                            
  4. 网关模式验证 — gateway.mode 是否已设置
  5. 认证模式歧义检测 — 同时配置 token 和 password 时发出警告                                                                                                                           
  6. 认证配置文件修复 — 修复 OAuth 配置、移除废弃认证、检查密钥链访问
  7. 网关令牌生成 — 无认证时提议生成随机令牌                                                                                                                                            
  8. 遗留状态迁移 — 迁移旧版会话、智能体状态、WhatsApp 认证                                                                                                                             
  9. 状态完整性 — 验证状态存储一致性                                                                                                                                                    
  10. 会话锁健康 — 检查陈旧的会话锁                                                                                                                                                     
  11. 沙箱镜像修复 — 修复/拉取 Docker 沙箱镜像                                                                                                                                          
  12. 网关服务扫描与修复                                                                                                                                                                
  13. 安全警告 — 高亮重要安全问题
  14. 浏览器就绪状态 — Chrome MCP 浏览器设置检查
  15. 网关健康探测与内存子系统探测                                                                                                                                                      
  16. Shell 补全安装
  17. 工作区状态与引导文件大小检查                                                                                                                                                      
                                                                                                                                                                                        
  支持交互式/非交互式/自动修复模式。 
```
