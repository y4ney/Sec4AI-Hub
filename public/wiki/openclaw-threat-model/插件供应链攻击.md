# 插件供应链攻击

#### 威胁描述

插件中的 npm 依赖项可能被入侵或本身就是恶意的。一个被入侵的依赖项可能影响该插件的所有用户并扩散到核心系统。

#### 缓解措施

对所有插件执行依赖审计。`patches/ `中的精确版本锁定防止供应链篡改。安全审计会标记未锁定的外部依赖。
