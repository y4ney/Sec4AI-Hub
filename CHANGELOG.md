## [1.0.0] - 2025-04-06

### Added
- OpenClaw MAESTRO threat model (39 threats across 7 layers)
- AI Agent threat model (17 threats + 6 defense playbooks)
- Interactive SPA with dark theme, global search, and breadcrumb navigation
- Docker deployment (CIS Benchmark compliant, multi-stage build)
- CI/CD pipeline (GitHub Actions: build, type-check, GitHub Pages deploy)
- Issue/PR templates, Contributing Guide, Code of Conduct, Security Policy
